Major Qualifying Project

Worcester Polytechnic Institute - 2021-2022

For my senior capstone project I worked worked on the design and analysis of a high power model rocket project. I worked independently to develop a 6DOF flight simulator in MATLAB.

I developed a 6DOF simulator including a oblate rotating earth model. I created the program in MATLAB using object oriented programming which significantly simplified use of the program to simulate various rockets.
